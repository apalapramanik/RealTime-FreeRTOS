# CSCE 892: Real-Time Systems - Spring 2024

A collection of lab assignments exploring real-time embedded systems programming using the Ringo Robot platform and FreeRTOS.

## Labs

### Lab 1: Introduction to Ringo Robot
Getting started with the Ringo Robot platform and basic embedded programming.

**Directory:** `Lab1_ApalaPramanik/`

### Lab 2: Real-Time Task Scheduling
Implementing real-time task scheduling concepts on embedded hardware.

**Directory:** `Lab2_ApalaPramanik/`

### Lab 3: FreeRTOS Task Management
Multi-task programming using FreeRTOS on the Ringo Robot:
- FreeRTOS task creation and scheduling
- Task handles and function prototypes
- Robot navigation and motor control
- PID control implementation (velocity control)
- Rotation and movement tasks

**Directory:** `Lab3_ApalaPramanik/`

## Technologies Used
- Arduino (C/C++)
- FreeRTOS (Real-Time Operating System)
- Ringo Robot Platform (Plum Geek)
- RingoHardware Library

## Hardware
- Ringo Robot by Plum Geek
- Arduino-compatible microcontroller
- Built-in navigation sensors
- Motor control system

## Key Concepts
- Real-time task scheduling
- FreeRTOS task management (`xTaskCreate`, `vTaskStartScheduler`)
- PID control for robot navigation
- Embedded systems programming
- Concurrent task execution
